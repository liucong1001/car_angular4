

# ngx-admin
1.环境,安装angular/cli
npm install -g cnpm  
cnpm install -g @angular/cli
安装好angular/cli之后，就可以用ng命令啦。
例如：ng --version 查看版本号
ng help 帮助

2.生成模块，组件
```
ng g m test --routing
ng  g c test/testapi --routing
```
外网访问工具下载地址 ：https://github.com/getlantern/forum/issues/833
